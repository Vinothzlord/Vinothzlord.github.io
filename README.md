# Vinoth Sankar M — Red Team Portfolio

A professional red team / cybersecurity portfolio built with pure HTML, CSS, and JavaScript.

## 🚀 Deploy on GitHub Pages

### Step 1 — Create a new repository
- Go to [github.com/new](https://github.com/new)
- Name it exactly: `Vinothzlord.github.io`
- Set it to **Public**
- Click **Create repository**

### Step 2 — Upload the files
- Click **Add file → Upload files**
- Drag and drop `index.html` and `README.md`
- Click **Commit changes**

### Step 3 — Enable GitHub Pages
- Go to **Settings → Pages**
- Under **Source**, select `Deploy from a branch`
- Branch: `main` / Folder: `/ (root)`
- Click **Save**

### Step 4 — Visit your live site
Your portfolio will be live at:
```
https://Vinothzlord.github.io
```
> It may take 1–2 minutes to go live after the first deploy.
